16 8月 2018

基于HDFS网盘速度测试

cd /mnt/yz-hadoop-hdfs/urs

使用命令dd测试写速度

urs@urs-risk0:/mnt/yz-hadoop-hdfs/urs$ time dd if=/dev/zero of=test.dbf bs=8k count=300000
300000+0 records in
300000+0 records out
2457600000 bytes (2.5 GB) copied, 30.9095 s, 79.5 MB/s

cd ~/ETL/tmp
urs@urs-risk0:~/ETL/tmp$ time dd if=/dev/zero of=test.dbf bs=8k count=300000
300000+0 records in
300000+0 records out
2457600000 bytes (2.5 GB) copied, 2.57429 s, 955 MB/s

测试读速度

urs@urs-risk0:/mnt/yz-hadoop-hdfs/urs$ time dd if=/mnt/yz-hadoop-hdfs/urs/test.dbf of=/dev/null bs=8k
300000+0 records in
300000+0 records out
2457600000 bytes (2.5 GB) copied, 27.2977 s, 90.0 MB/s

real    0m27.528s
user    0m0.048s
sys    0m1.124s

urs@urs-risk0:~/ETL/tmp$ time dd if=test.dbf of=/dev/null bs=8k

300000+0 records out
